# Lesson Pack Generator for Chinese Classrooms

Generate classroom-ready lesson assets from one natural-language teaching brief.

## Best for

- one-lesson prep packs
- lesson outlines
- intros and hook ideas
- in-class practice tasks
- homework and extension tasks

## What makes this useful

- built around Chinese K-12 lesson preparation habits
- starts from one teaching seed instead of a long form
- organizes outputs in a classroom-ready order
- balances hard constraints with optional soft preferences

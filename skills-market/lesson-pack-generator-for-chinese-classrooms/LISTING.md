## About

Many lesson-generation prompts produce long, vague output that still needs major cleanup before a teacher can actually use it.

This skill is built around a more practical pattern: start with one natural-language teaching brief, extract the hard constraints and soft preferences, and generate a compact lesson-ready output in a usable order.

This package includes:

- a lesson-pack workflow `SKILL.md`
- templates for lesson brief extraction and output ordering
- classroom-oriented examples for reading and math lessons

Best for:

- one-lesson prep packs
- lesson outlines
- intro ideas
- in-class practice
- homework and extension tasks

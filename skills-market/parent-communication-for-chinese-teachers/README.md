# Parent Communication for Chinese Teachers

Turn one messy teacher note into calm, usable parent communication for Chinese K-12 contexts.

## Best for

- parent follow-up messages
- class notices
- homework reminders
- habit and behavior communication
- learning suggestions

## What makes this useful

- built around real Chinese school communication patterns
- optimized for warm but clear teacher tone
- avoids generic customer-support-style writing
- includes reusable templates and example scenarios

## About

Most parent communication tools get the tone wrong. They sound too corporate, too stiff, or too vague to send in a real class group or one-to-one parent message.

This skill is built around real Chinese K-12 communication patterns. Give your agent one natural-language brief like “the student has been distracted in class and delaying homework, write a gentle but clear message for the parent,” and it will shape that into a usable teacher-style message.

This package includes:

- a communication-focused `SKILL.md`
- examples for common school scenarios
- reusable templates for class notices and gentle parent follow-ups

Best for:

- class group notices
- behavior or habit follow-ups
- homework reminders
- academic progress check-ins
- parent communication that should feel warm, specific, and professional
